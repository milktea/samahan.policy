samahan.policy Installation
---------------------------

To install samahan.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``samahan.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        samahan.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
